    Naruto Uzumaki (Pre-Timeskip)¶

    Name: Naruto Uzumaki

    Version: Pre-Timeskip

    Origin: Naruto

    Roles: Summoner, Striker

    Difficulty: High

    Gameplan: Open with Shadow Clone Jutsu to flood your hand with disposable clone attacks and bank Shadow Clones as a resource. Keep 1 clone in reserve to turn Defend into an Evade. Convert the swarm into Naruto Uzumaki Barrage to win zones via Multihit pressure, then cash out with a clone-assisted Rasengan. When Nine-Tails' Resentment triggers, Kyuubi Chakra and One-Tail Cloak fuel Gamabunta follow-ups and unlock One-Tail Rasengan to close.

    Innates¶

    Nine-Tails' Resentment

    The first time this character reaches 25% HP, gain 4 Kyuubi Chakra and +3 Kyuubi Chakra Count. Gain 4 One-Tail Cloak. Gain 2 Shadow Clones.

    Status Effects¶

    Shadow Clones

    Type Unique.

    Max Stack 6.

    Turn End Reduce Stack by 1.

    Effect Used to enhance Naruto's cards.

    Kyuubi Chakra

    Type Unique.

    Potency Max 4.

    Count Max 4.

    Turn Start Gain X Strength. Gain X Haste.

    Effect The first time each turn this character plays Shadow Clone Jutsu, gain X Shadow Clones. X equals Potency.

    Turn End Reduce Count by 1. Reduce current HP by X times 5%. X equals Potency.

    Summoned: Gamabunta

    Type Unique.

    Max Stack 3.

    Allies' Card Played Once per turn, if Stack is 1+ and any ally plays an Attack that costs 2+ Energy and targets an enemy: Create 1 Gamabunta: Toad Smash in this character's hand.

    Turn End Reduce Stack by 1.

    One-Tail Cloak

    Type Unique.

    Max Stack 4.

    Effect A marker used to unlock Naruto's low-HP finishers.

    Turn End Reduce Stack by 1.

    Cards¶

    Card 1: Strike¶

    Cost: 1 Energy Power: 8-12 Type: Basic, Attack, Physical, Melee, Blunt Target: 1 Enemy Speed: Normal

    Effect

    Deal Power damage.

    On Hit: If this character has Kyuubi Chakra, gain 1 Shadow Clones.

    Card 2: Defend¶

    Cost: 1 Energy Power: 8-12 Type: Basic, Defense, Physical, Melee Target: 1 Ally Speed: Normal

    Effect

    Gain Power Shield.

    If this character has 1+ Shadow Clones: You may spend 1 Shadow Clones. If you do, this card gains Evade.

    Card 3: Shadow Clone Jutsu¶

    Cost: 1 Energy Power: - Type: Technique, Special, Magical Target: Self Speed: Fast

    Effect

    Gain 2 Shadow Clones.

    Create 2 Shadow Clone: Strike in hand.

    Innate.

    Card 4: Naruto Uzumaki Barrage¶

    Cost: 2 Energy Power: 16-24 Type: Technique, Attack, Physical, Melee, Blunt, Multihit Target: 1 Enemy Speed: Normal

    Effect

    Choose X (0-3).

    Spend X Shadow Clones.

    Deal Power / (X + 2) damage (X + 2) times.

    On Hit: If X is 3, inflict 1 Stun.

    Card 5-1: Rasengan¶

    Cost: 3 Energy Power: 24-36 Type: Technique, Attack, Magical, Melee, Blunt Target: 1 Enemy Speed: Slow

    Effect

    Deal Power damage.

    If this character has 2+ Shadow Clones: This card becomes Rasengan: Clone Assist.

    Card 5-2: Rasengan: Clone Assist¶

    Cost: 3 Energy Power: 24-36 Type: Technique, Attack, Magical, Melee, Blunt Target: 1 Enemy Speed: Normal

    Effect

    Spend 2 Shadow Clones.

    Follow-Up.

    Deal Power damage.

    On Hit: Inflict 2 Vulnerable and +2 Vulnerable Count.

    Follow-Up.

    Deal Power damage.
