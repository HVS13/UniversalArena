    Kurosaki Ichigo (Soul Society)¶

    Name: Kurosaki Ichigo

    Version: Soul Society

    Origin: Bleach

    Roles: Striker, Skirmisher

    Difficulty: Medium

    Gameplan: Win fights the way Ichigo does here: clean sword pressure, Shunpo to steal tempo, build Reiatsu through exchanges, then cash out with Getsuga. Bankai is your end-it speed window, and Hollow Interference is a single risky surge that can save or throw the fight.

    Innates¶

    Fighting Spirit

    The first time each turn this character takes damage to HP from an enemy Attack, gain 1 Reiatsu.

    Inner Hollow (Uncontrolled)

    The first time this character reaches 25% HP, gain 1 Hollow Interference. Gain 2 Reiatsu.

    Status Effects¶

    Reiatsu

    Type Unique.

    Max Value 6.

    Turn End Reduce Value by 1.

    Effect A resource gained through combat exchanges; spent to amplify Getsuga Tensho.

    Bankai: Tensa Zangetsu

    Type Unique.

    Count Max 6.

    Turn Start Gain 2 Haste. Gain 1 Strength.

    Effect The first time each turn this character hits with an Attack, gain 1 Reiatsu.

    Turn End Reduce Count by 1.

    When this expires Gain 1 Strain and +3 Strain Count.

    Hollow Interference

    Type Unique.

    Max Stack 1.

    Turn Start Gain 2 Haste. Gain 2 Strength.

    Turn End Expires.

    When this expires Gain 1 Strain and +3 Strain Count.

    Card 1-1: Strike¶

    Cost: 1 Energy Power: 10-14 Type: Basic, Attack, Physical, Melee, Slash Target: 1 Enemy Speed: Normal

    Effect

    Deal Power damage. On Hit: Gain 1 Reiatsu.

    If this character has Hollow Interference: This card becomes Strike: White.

    Otherwise, if this character has Bankai: Tensa Zangetsu: This card becomes Strike: Tensa Zangetsu.
