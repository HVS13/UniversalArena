    Rover (Spectro)¶

    Name: Rover

    Version: Spectro

    Origin: Wuthering Waves

    Difficulty: Low

    Gameplan: Play as many cards as possible to fill Ultimate Meter faster. Gain Forte to inflict Spectro Frazzle stacks. Heal using ultimate.

    Innates¶

    World in a Grain of Sand

    Gain 1 Ultimate Meter each time this character card is played.

    Status Effects¶

    Forte: Rover Spectro

    Type Neutral.

    Max Value 2.

    Effect Used to enhance some cards.

    Spectro Frazzle

    Type Negative.

    Max Stack 10.

    Turn End Inflict damage equal to 5 x Stack. Then reduce Stack by 1.

    Stagnate

    Type Negative.

    Max Value 10.

    Instant Increase the Energy cost of a random card in hand by 1 until that card is used. Repeat Value times. Expires.
